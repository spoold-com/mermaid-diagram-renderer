Create and preview Mermaid diagrams online. Supports flowcharts, sequence diagrams, class diagrams, ER diagrams, Gantt charts, and more.

## Features

- Live render of Mermaid syntax as you type
- Flowcharts, sequence, class, ER, Gantt, and other diagram types
- Export as PNG or SVG
- Syntax highlighting and error hints
- No install; same syntax as GitHub and GitLab

## Why use this tool

Sketch architecture, flows, or sequences in text and see the diagram immediately. Perfect for READMEs and docs that use Mermaid.
